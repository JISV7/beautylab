Datatable types
**Name**: Summary

## Buttons

- **button-group-selector**: Button group selector (multiple button instances)
- **button-selector**: Button selector

## DataTables

- **cell-selector**: Selector options for cells.
- **column-selector**: Selector options for columns.
- **DataTables.Api**: DataTables API object instance
- **DataTables.Order**: DataTables ordering object
- **DataTables.SearchOptions**: DataTables search options object
- **DataTables.Settings**: DataTables settings object
- **DataTables.Type**: DataTables data type object
- **row-selector**: Selector options for rows.
- **selector-modifier**: Options for how the row, column and cell selector should operate on rows.
- **table-selector**: Selector options for tables.

## Javascript

- **array**: JavaScript Array type
- **boolean**: JavaScript Boolean type
- **function**: JavaScript Function
- **integer**: Subset of the JavaScript Number type using only integers
- **jQuery**: jQuery object instance
- **node**: DOM element
- **null**: JavaScript null type
- **number**: JavaScript Number type
- **object**: JavaScript Object type
- **string**: JavaScript String type
- **undefined**: JavaScript undefined type

## SearchBuilder

- **SearchBuilder.Condition**: Search condition plug-in structure
- **SearchBuilder.Criteria**: Object that describes a search condition

## StateRestore

- **stateRestoreConfig**: Configuration options for StateRestore button
